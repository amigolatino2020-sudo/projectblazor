# Blazor Task Manager - Microsoft Copilot Demonstration

## Project Overview
This project demonstrates the use of Microsoft Copilot for:
- **Activity 1**: Code generation for Blazor components, services, and models
- **Activity 2**: Debugging common issues and performance optimization
- **Activity 3**: Building a complete, production-ready Blazor application

## Technology Stack
- .NET 8/9
- Blazor Server Interactive
- Bootstrap 5
- C# 12

## Features
- ✅ Complete CRUD operations for task management
- ✅ Task filtering and searching
- ✅ Priority-based color coding
- ✅ Responsive design with Bootstrap
- ✅ Async/await patterns throughout

## How to Run
1. Install .NET 8 SDK
2. Clone this repository
3. Run `dotnet restore`
4. Run `dotnet run --project src/BlazorCopilotProject`
5. Navigate to `https://localhost:5001/tasks`
